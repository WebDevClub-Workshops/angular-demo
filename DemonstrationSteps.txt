Angular Demo

1. Install Node (includes npm)
2. Show how to check Node.js and npm version + briefly talk about the benefits of NVM / NPM in general
3. Install Angular
	- `npm install -g @angular/cli`
	- If windows gets a PS policy error: `Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned`
	- If on unix, may need to do `sudo npm install -g @angular/cli`
4. Run `ng new my-app`
5. Show how to run, Explain Component structure, App structure, etc.
6. Generate new Home and about component with `npm install g c home-component` and `npm install g c about-component` in the app directory
7. Show how to add components to Routing
